[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

這是一個強大的網頁工具，專門設計用於去除由 Google Gemini 生成圖片中的浮水印。此工具完全在瀏覽器端運行，無需將圖片上傳至伺服器，確保您的隱私安全。

## 🎯 與原版的差異
本專案是專注於浮水印移除並強化標誌自訂功能的**改版**。

## ✨ 主要功能

- **🚫 自動去除浮水印**：利用逆向 Alpha 混合演算法（Reverse Alpha Blending），搭配多維度自動強度偵測，精確還原被浮水印覆蓋的像素。
  - 搜尋範圍：0.05 ~ 2.0（優化自 0.1 ~ 1.2）
  - 搜尋精度：0.01（提升一倍）
  - 多維度評估：亮度一致性、邊緣融合度、峰值偵測
- **🎨 智慧 Logo 疊加**：上傳您的 Logo 圖片，自動對齊浮水印位置進行智慧覆蓋：
  - **自動定位**：Logo 會自動對齊偵測到的浮水印中心位置
  - **尺寸一致**：Logo 大小基於圖片較短邊計算（橫式、直式輸出尺寸一致）
  - **可調整設定**：透明度 0% ~ 100%（預設 20%）、大小 10% ~ 300%（預設 200%）
- **🔧 輸出選項**：
  - **輸出縮小**：1280×720 / 1920×1080（或關閉）
  - **保留 EXIF**：保留原始圖片元數據
  - **銳化**：套用銳化濾鏡提升視覺品質
  - **檔名前綴**：自訂輸出檔名前綴
  - **排序**：依檔名或時間排序，支援升冪/降冪
- **📥 彈性下載選項**：
  - **三種下載類型**：
    - **一般版 (R_)**：去浮水印 + 縮小 + 含 Logo
    - **鏡射版 (M_)**：去浮水印 + 縮小 + 水平鏡像（無 Logo）
    - **純淨版 (N_)**：去浮水印 + 縮小（無 Logo）
  - **單張下載**：點擊下載按鈕取得各版本
  - **批次下載**：ZIP 會同時包含 R_ 和 N_ 兩種版本
- **⚙️ 浮水印位置設定**：
  - **自動偵測**：根據圖片自動判斷浮水印位置
  - **新版邊距**：適用於 192px（橫式）/ 96px（直式）浮水印
  - **舊版邊距**：適用於 64px（橫式）/ 32px（直式）浮水印
- **🔒 隱私優先**：所有處理皆在您的本地瀏覽器中完成，圖片不會離開您的裝置。
- **⚡ 即時預覽**：上傳即處理，快速查看結果。
- **🖱️ 拖拽支援**：支援將圖片直接拖拽至視窗進行處理。
- **👀 對比模式**：長按處理後的圖片即可查看原始圖片，方便比較去除效果。
- **💡 Lightbox 燈箱模式**：點擊圖片開啟全螢幕檢視，支援左右箭頭導航和鍵盤操作
  - **縮放控制**：滾輪放大/縮小（50%~500%），支援拖曳平移
  - **鏡射預覽**：水平/垂直鏡射，快速確認去除效果
  - **旋轉控制**：90° 旋轉，方便確認各角度效果
  - **長按原圖**：長按顯示原圖進行對比
- **🌙 深色/淺色主題**：一鍵切換深色或淺色介面主題。
- **💾 高畫質下載**：支援 PNG（無損）或 JPEG（壓縮）格式。
- **📋 剪貼簿貼上**：支援直接貼上 (Ctrl+V) 截圖或圖片進行處理。
- **📦 批次 ZIP 下載**：下載多張圖片時自動打包為 ZIP 檔案（`banana_watermark_remover.zip`）。
- **🌐 多國語系支援**：介面支援英文、繁體中文、簡體中文、日文及韓文。

## 🛠️ 技術原理

此專案使用純 JavaScript (Canvas API) 實作。它預先載入了 Gemini 浮水印的 Alpha 遮罩（Mask），並通過計算每個像素的原始顏色值來「反算」扣除浮水印的影響，從而達到無損或近乎無痕的去除效果。

## 🚀 如何使用

1. **開啟網頁**：直接在瀏覽器中打開 `index.html`。
2. **上傳圖片**：點擊上傳區域選擇圖片，或直接將 JPG/PNG/WEBP 圖片拖入。
3. **查看結果**：系統會自動處理並顯示結果。
4. **調整設定**（如有需要）：
   - 切換「自動偵測」/「新版邊距」/「舊版邊距」調整浮水印位置
   - 切換「自動」/「強制小」/「強制大」調整浮水印大小
   - 手動拖曳強度滑桿微調去除強度
5. **下載**：
   - **單張下載**：點擊「一般」(R_)/「鏡射」(M_)/「純淨」(N_) 按鈕下載
   - **批次下載**：點擊「下載全部」按鈕

## 📦 安裝與執行

本專案為靜態網頁，無需安裝複雜的後端環境。

1. **複製專案**：
   ```bash
   git clone https://github.com/aflypenstudio/BananaWatermarkRemover.git
   ```
2. **進入目錄**：
   ```bash
   cd BananaWatermarkRemover
   ```
3. **執行**：
   直接用瀏覽器打開 `index.html` 即可使用。
   *注意：由於瀏覽器安全策略（CORS），若直接開啟本地檔案可能會導致遮罩圖片載入失敗。建議使用簡單的本地伺服器運行，例如使用 Python：*
   ```bash
   # Python 3
   python -m http.server 8000
   ```

   然後在瀏覽器訪問 `http://localhost:8000`
