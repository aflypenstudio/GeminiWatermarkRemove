[English](README.md) | [繁体中文](README_zh-TW.md) | [简体中文](README_zh-CN.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

这是一个强大的网页工具，专门设计用于去除由 Google Gemini 生成图片中的水印。此工具完全在浏览器端运行，无需将图片上传至服务器，确保您的隐私安全。

## 🎯 与原版的差异
本项目是专注于水印移除并强化标志定制功能的**改版**。

## ✨ 主要功能

- **🚫 自动去除水印**：利用逆向 Alpha 混合算法（Reverse Alpha Blending），搭配多维度自动强度检测，精确还原被水印覆盖的像素。
  - 搜索范围：0.05 ~ 2.0（优化自 0.1 ~ 1.2）
  - 搜索精度：0.01（提升一倍）
  - 多维度评估：亮度一致性、边缘融合度、峰值检测
- **🎨 智能 Logo 叠加**：上传您的 Logo 图片，自动对齐水印位置进行智能覆盖：
  - **自动定位**：Logo 会自动对齐侦测到的水印中心位置
  - **尺寸一致**：Logo 大小基于图片较短边计算（横向、竖向输出尺寸一致）
  - **可调整设定**：透明度 0% ~ 100%（预设 20%）、大小 10% ~ 300%（预设 200%）
- **🔧 输出选项**：
  - **输出缩小**：1280×720 / 1920×1080（或关闭）
  - **保留 EXIF**：保留原始图片元数据
  - **锐化**：套用锐化滤镜提升视觉品质
  - **文件名前缀**：自定义输出文件名前缀
  - **排序**：依文件名或时间排序，支持升序/降序
- **📥 弹性下载选项**：
  - **三种下载类型**：
    - **一般版 (R_)**：去水印 + 缩小 + 含 Logo
    - **镜像版 (M_)**：去水印 + 缩小 + 水平镜像（无 Logo）
    - **纯净版 (N_)**：去水印 + 缩小（无 Logo）
  - **单张下载**：点击下载按钮取得各版本
  - **批量下载**：ZIP 会同时包含 R_ 和 N_ 两种版本
- **⚙️ 水印位置设定**：
  - **自动检测**：根据图片自动判断水印位置
  - **新版边距**：适用于 192px（横向）/ 96px（竖向）水印
  - **旧版边距**：适用于 64px（横向）/ 32px（竖向）水印
- **🔒 隐私优先**：所有处理皆在您的本地浏览器中完成，图片不会离开您的设备。
- **⚡ 即时预览**：上传即处理，快速查看结果。
- **🖱️ 拖拽支持**：支持将图片直接拖拽至窗口进行处理。
- **👀 对比模式**：长按处理后的图片即可查看原始图片，方便比较去除效果。
- **💡 Lightbox 灯箱模式**：点击图片开启全屏检视，支持左右箭头导航和键盘操作
  - **缩放控制**：滚轮放大/缩小（50%~500%），支持拖拽平移
  - **镜像预览**：水平/垂直镜像，快速确认去除效果
  - **旋转控制**：90° 旋转，方便确认各角度效果
  - **长按原图**：长按显示原图进行对比
- **🌙 深色/浅色主题**：一键切换深色或浅色界面主题。
- **💾 高画质下载**：支持 PNG（无损）或 JPEG（压缩）格式。
- **📋 剪贴板粘贴**：支持直接粘贴 (Ctrl+V) 截图或图片进行处理。
- **📦 批量 ZIP 下载**：下载多张图片时自动打包为 ZIP 文件（`banana_watermark_remover.zip`）。
- **🌐 多语言支持**：界面支持英文、繁体中文、简体中文、日文及韩文。

## 🛠️ 技术原理

此项目使用纯 JavaScript (Canvas API) 实现。它预先加载了 Gemini 水印的 Alpha 遮罩（Mask），并通过计算每个像素的原始颜色值来"反算"扣除水印的影响，从而达到无损或近乎无痕的去除效果。

## 🚀 如何使用

1. **开启网页**：直接在浏览器中打开 `index.html`。
2. **上传图片**：点击上传区域选择图片，或直接将 JPG/PNG/WEBP 图片拖入。
3. **查看结果**：系统会自动处理并显示结果。
4. **调整设定**（如有需要）：
   - 切换「自动检测」/「新版边距」/「旧版边距」调整水印位置
   - 切换「自动」/「强制小」/「强制大」调整水印大小
   - 手动拖曳强度滑杆微调去除强度
5. **下载**：
   - **单张下载**：点击「一般」(R_)/「镜像」(M_)/「纯净」(N_) 按钮下载
   - **批量下载**：点击「下载全部」按钮

## 📦 安装与执行

本项目为静态网页，无需安装复杂的后端环境。

1. **复制项目**：
   ```bash
   git clone https://github.com/aflypenstudio/BananaWatermarkRemover.git
   ```
2. **进入目录**：
   ```bash
   cd BananaWatermarkRemover
   ```
3. **执行**：
   直接用浏览器打开 `index.html` 即可使用。
   *注意：由于浏览器安全策略（CORS），若直接开启本地文件可能会导致遮罩图片加载失败。建议使用简单的本地服务器运行，例如使用 Python：*
   ```bash
   # Python 3
   python -m http.server 8000
   ```

   然后在浏览器访问 `http://localhost:8000`
