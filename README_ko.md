# 바나나 워터마크 제거기 & 조정기

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

[English](README.md) | [繁체中文](README_zh-TW.md) | [简体中文](README_zh-CN.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

Google Gemini가 생성한 이미지에서 워터마크를 제거하도록 설계된 강력한 웹 도구입니다. 이 도구는 브라우저에서 완전히 실행되므로 이미지가 서버에 업로드되지 않아 개인 정보가 보호됩니다.

## 🎯 원본과의 차이점
이 프로젝트는 워터마크 제거에 특화된**수정 버전**으로 로고 사용자 지정 기능이 강화되어 있습니다.

## ✨ 주요 기능

- **🚫 자동 워터마크 제거**: 역 알파 블렌딩 알고리즘(Reverse Alpha Blending)과 다차원 자동 강도 감지를 사용하여 워터마크로 덮인 픽셀을 정밀하게 복원합니다.
  - 검색 범위: 0.05 ~ 2.0 (기존: 0.1 ~ 1.2)
  - 검색 정밀도: 0.01 (기존 대비 2배 향상)
  - 다차원 평가: 밝기 균일성, 엣지 블렌딩, 피크 감지
- **🎨 스마트 로고 오버레이**: 로고 이미지를 업로드하면 워터마크 위치에 자동으로 배치됩니다:
  - **자동 위치**: 로고가 감지된 워터마크 위치에 자동으로 배치됩니다
  - **일관된 크기**: 로고 크기는 이미지의 짧은 변을 기준으로 계산됩니다 (가로/세로 같은 크기)
  - **조정 가능**: 투명도 0%~100% (기본값 20%), 크기 10%~300% (기본값 200%)
- **🔧 출력 옵션**:
  - **출력 리사이즈**: 1280×720 / 1920×1080 (또는 해제)
  - **EXIF 유지**: 원본 이미지 메타데이터 유지
  - **선명도**: 선명도 필터 적용
  - **파일명 접두사**: 출력 파일명 사용자 지정
  - **정렬**: 파일명 또는 날짜순으로 정렬 (오름차순/내림차순)
- **📥 유연한 다운로드 옵션**:
  - **다운로드 유형**:
    - **보통版 (R_)**: 워터마크 제거 + 리사이즈 + 로고 포함
    - **미러版 (M_)**: 워터마크 제거 + 리사이즈 + 수평 미러링 (로고 없음)
    - **클린版 (N_)**: 워터마크 제거 + 리사이즈 (로고 없음)
  - **단일 이미지**: 다운로드 버튼으로 각 버전 다운로드
  - **일괄 다운로드**: ZIP에 R_과 N_ 버전이 모두 포함됨
- **⚙️ 워터마크 위치 설정**:
  - **자동 감지**: 이미지에 따라 워터마크 위치를 자동으로 판단
  - **새 마진**: 192px (가로) / 96px (세로) 워터마크용
  - **이전 마진**: 64px (가로) / 32px (세로) 워터마크용
- **🔒 개인 정보 우선**: 모든 처리는 로컬 브라우저에서 완료되며 이미지는 기기를 떠나지 않습니다.
- **⚡ 즉시 미리보기**: 업로드 즉시 처리되어 결과를 빠르게 확인할 수 있습니다.
- **🖱️ 드래그 앤 드롭 지원**: 이미지를 창으로 직접 드래그하여 처리할 수 있습니다.
- **👀 비교 모드**: 처리된 이미지를 길게 누르면 원본 이미지를 볼 수 있어 제거 효과를 쉽게 비교할 수 있습니다.
- **💡 라이트박스 모드**: 이미지를 클릭하여 전체 화면 보기를 열고 화살표 키 또는 키보드로 이동 가능
  - **확대/축소 제어**: 스크롤 휠로 확대/축소 (50%~500%), 확대 시 드래그로 이동 가능
  - **미러링 미리보기**: 수평/수직 플립으로 효과 빠르게 확인
  - **회전 제어**: 90° 회전으로 모든 각도에서 확인 가능
  - **원본 길게 누르기**: 길게 눌러 원본과 비교
- **🌙 다크/라이트 테마**: 원클릭으로 다크 테마와 라이트 테마를 전환.
- **💾 고화질 다운로드**: PNG(무손실) 또는 JPEG(압축) 형식으로 다운로드 가능.
- **📋 클립보드 붙여넣기**:スクリーン샷이나 이미지를 직접 붙여넣기(Ctrl+V)하여 처리할 수 있습니다.
- **📦 일괄 ZIP 다운로드**:複数の이미지를 다운로드할 때 자동으로 ZIP 파일로 묶어줍니다.
- **🌐 다국어 지원**: 인터페이스는 영어,繁体 중국어,简体中文,일본어,한국어를 지원합니다.

## 🛠️ 기술 원리

이 프로젝트는 순수 JavaScript(Canvas API)로 구현되었습니다. Gemini 워터마크의 알파 마스크를 미리 로드하고 각 픽셀의 원본 색상 값을 계산하여 워터마크의 영향을 "역산"함으로써 무손실 또는 흔적 없는 제거 효과를 달성합니다.

## 🚀 사용 방법

1. **페이지 열기**: 브라우저에서 `index.html`을 직접 엽니다.
2. **이미지 업로드**: 업로드 영역을 클릭하거나 JPG/PNG/WEBP 이미지를 드래그합니다.
3. **결과 확인**: 시스템이 자동으로 처리하고 결과를 표시합니다.
4. **설정 조정** (필요한 경우):
   - «자동»/«새 마진»/«이전 마진»을 전환하여 워터마크 위치 조정
   - «자동»/«소형 강제»/«대형 강제»를 전환하여 워터마크 크기 조정
   - 강도 슬라이더를 수동으로 드래그하여 제거 강도 미세 조정
5. **다운로드**:
   - **단일 이미지**: «보통»(R_) / «미러»(M_) / «클린»(N_) 버튼을 클릭하여 다운로드
   - **일괄 다운로드**: «모두 다운로드» 버튼을 클릭

## 📦 설치 및 실행

이 프로젝트は 정적 웹 페이지이므로 복잡한 백엔드 환경을 설치할 필요가 없습니다.

1. **프로젝트 클론**:
   ```bash
   git clone https://github.com/aflypenstudio/BananaWatermarkRemover.git
   ```
2. **디렉토리 이동**:
   ```bash
   cd BananaWatermarkRemover
   ```
3. **실행**:
   브라우저에서 `index.html`을 직접 열어 사용할 수 있습니다.
   *주의: 브라우저 보안 정책(CORS)으로 인해 로컬 파일을 직접 열면 마스크 이미지 로드에 실패할 수 있습니다. Python 등을 사용하여 간단한 로컬 서버를 실행하는 것이 좋습니다:*
   ```bash
   # Python 3
   python -m http.server 8000
   ```

   그런 다음 브라우저에서 `http://localhost:8000`에 접속하십시오.
