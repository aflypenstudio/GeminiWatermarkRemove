# 更新日誌 (Changelog)

所有重要的更新都會記錄在這裡。

## [0.7] - 2026-07-10

### 💡 Lightbox 功能增強
- **縮放控制**：
  - 滑鼠滾輪放大/縮小（50% ~ 500%）
  - 按鈕快速放大 (+) / 縮小 (-) / 重置 (100%)
  - 拖曳平移（僅在縮放 > 100% 時啟用）
  - 雙指縮放（觸控裝置）
- **鏡射預覽**：
  - 水平鏡射 (H 鍵) - 快速預覽圖片水平翻轉效果
  - 垂直鏡射 (V 鍵) - 快速預覽圖片垂直翻轉效果
  - 按鈕狀態顯示開/關
- **旋轉控制**：
  - 向左旋轉 90° (Q 鍵)
  - 向右旋轉 90° (E 鍵)
  - 配合鍵盤方向鍵導航使用
- **鍵盤快捷鍵**：
  - `+` / `=` 放大
  - `-` 縮小
  - `0` 重置縮放
  - `H` 水平鏡射
  - `V` 垂直鏡射
  - `Q` 向左旋轉
  - `E` 向右旋轉
  - `←` / `→` 上一張/下一張
  - `Esc` 關閉
- **長按原圖對比**：依舊支援長按顯示原圖進行對比

### 📝 文件更新
- 更新多語系翻譯（新增縮放/鏡射相關翻譯）

## [0.6] - 2026-07-10

### ✨ 新功能
- **浮水印位置設定**：新增三種浮水印位置模式
  - **自動偵測**：根據圖片自動判斷浮水印位置
  - **新版邊距**：適用於 192px（橫式）/ 96px（直式）浮水印
  - **舊版邊距**：適用於 64px（橫式）/ 32px（直式）浮水印
- **鏡射下載 (M_)**：新增第三種下載類型，水平鏡像處理後的圖片（無 Logo）
- **Lightbox 燈箱模式**：點擊圖片開啟全螢幕檢視，支援左右箭頭導航和鍵盤操作
- **深色/淺色主題**：一鍵切換深色或淺色介面主題

### 🛠️ 技術更新
- 更新翻譯檔案，修正日文和韓文的亂碼問題
- 統一所有語系的翻譯命名（banana_clean、輸出品項等）

### 📝 文件更新
- 更新所有 README 文件，加入新功能說明
- 詳細說明三種下載類型（R_/M_/N_）
- 新增浮水印位置設定說明
